# Placement Preparation Tracker

A web application to help students organize and monitor their placement preparation.

## Features

- Login Page
- Dashboard
- Coding Tracker
- Project Tracker
- Aptitude Tracker
- Company Tracker
- Local Storage Support

## Technologies Used

- HTML
- CSS
- JavaScript
- Local Storage

## Future Enhancements

- Spring Boot Backend
- MySQL Database
- User Authentication
- Dashboard Statistics
